Instructions for Uploading Colleges

To add more data to the college database, create an excel sheet with the following columns in order:

  1. College Name
  2. College description
  3. Regular Deadline (M-D-Y) (Excel should autoformat dates correctly)
  4. Early Deadline (M-D-Y) (Excel should autoformat dates correctly)
  5. Scholarship Deadline (M-D-Y) (Excel should autoformat dates correctly)
  6. FAFSA Deadline (M-D-Y) (Excel should autoformat dates correctly)
  7. Acceptance Announcement Date (M-D-Y) (Excel should autoformat dates correctly)
  8. Image URL

Enter data as in the sample image above in an excel file. When you are finished entering all the data, save your excel sheet as a CSV file, then upload it to this portal and click "Submit."
